Docker 方式安装

快速安装

$ git clone https://github.com/big-data-europe/docker-hadoop.git
$ cd docker-hadoop
# 选择版本
$ git tag --list
# 切换到制定版本
$ git branch hadoop-3.2.1 2.0.0-hadoop3.2.1-java8
$ git checkout hadoop-3.2.1

# 启动
$ docker-compose up -d

# 退出并删除容器
# $ docker-compose down
